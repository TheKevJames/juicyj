Rust Installation
~~~~~~~~~~~~~~~~~

::

    curl -sSf https://sh.rustup.rs | sh
    # complete installation process
    source ~/.cargo/env
    rustup toolchain install 1.5.0
    rustup default 1.5.0
    rustc --version  # should show "rustc 1.5.0 (3d7cd77e4 2015-12-04)"
