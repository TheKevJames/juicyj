Rust Installation
~~~~~~~~~~~~~~~~~

::

    curl -sSf https://sh.rustup.rs | sh
    # complete installation process
    source ~/.cargo/env
    rustup toolchain install 1.14.0
    rustup default 1.14.0
    rustc --version  # should show "rustc 1.14.0 (e8a012324 2016-12-16)"
